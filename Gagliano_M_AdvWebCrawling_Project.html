https://www.indeed.com/cmp/Kansas-State-University/reviews

The website was chosen due to the familiarity with the context of the data, and the ability to more effectively sift through the noise/irrelevant data. I chose Indeed.com instead of the increasingly-popular Glassdoor.com due to Glassdoor already indicating if the review is Positive/Negative based on the posted "Recommends/Does Not Recommend" feature.

alldata =[]

web = "https://www.indeed.com/cmp/Kansas-State-University/reviews?start=%s" #20 reviews per page == 10 pages

alldata =[]

for page in range(0, 200, 20):
    url = web % page
    data = html.fromstring(req.get(url).text)
#xpath
    for i in data.xpath('//div[@class="cmp-review"]'):
        name = i.xpath('div/div/span/span[@class="cmp-reviewer"]/text()')
        location = i.xpath('div/div/span[@class="cmp-reviewer-job-location"]/text()')
        
##-- SEE NOTES BELOW ABOUT THE FOLLOWING XPATH
        rating = i.xpath('div/div/span/meta/@content')
##-- Rating was not populating all 210 values from the original xpath for unknown reasons.
##-- Calling a more general xpath pulled all numerical values in correct indexed positions,\
##-- but additional cleanup will be necessary.

        review = i.xpath('div/div/span[@class="cmp-review-text"]/text()')
        date = i.xpath('div/div/span[@class="cmp-review-date-created"]/text()')
        #print(name, location, rating, review, date)
        alldata.append([name, location, rating, review, date])
